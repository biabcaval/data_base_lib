# Projeto de Banco de Dados - Biblioteca

## About the project
Nosso projeto é uma aplicação web de uma biblioteca com banco de dados MySQL, focando em operações CRUD e utilizando dados de 2 tabelas.

## Stacks utilizadas
1. Flask/Python
2. HTML
3. CSS
4. JavaScript
5. MySQL
